# Tic-Tac-Toe

## Descripción
Este proyecto es una implementación del clásico juego "Tres en Raya" (Tic-Tac-Toe). Permite a los jugadores competir en distintos modos de juego, incluyendo una versión imbatible basada en algoritmos estratégicos. Además, se está trabajando en una interfaz más intuitiva y en una IA avanzada que mejore la experiencia de juego.

## Características
- Modo Clásico: Dos jugadores pueden turnarse para jugar en una cuadrícula 3x3.
- Modo Imbatible: Un desafiante modo contra una IA optimizada (en desarrollo).
- Modo One Player: Un jugador contra una IA básica.
- Diseño visual atractivo: Personalización con archivos CSS.
- Interfaz en Desarrollo: El archivo `index.html` representa una versión en desarrollo de la interfaz principal del juego, que busca ofrecer una mejor experiencia de usuario.
- IA avanzada en desarrollo: Se está trabajando en un sistema de inteligencia artificial más desafiante y sofisticado.

## Tecnologías Utilizadas
- **HTML5:** Para la estructura del juego.
- **CSS3:** Para el diseño y estilizado visual.
- **JavaScript:** Para la lógica del juego e interactividad.

## Instalación
Sigue estos pasos para instalar y ejecutar el juego en tu equipo:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/ZAKrA529/tic-tac-toe.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd tic-tac-toe-main
   ```
3. Abre `index.html` en tu navegador para comenzar a jugar.

## Estructura del Proyecto
El proyecto está organizado de la siguiente manera:

- index.html: Archivo principal de la interfaz del juego (en desarrollo).
- gato.html: Otra versión del juego.
- imbatible.html: Modo de juego contra una IA avanzada (en desarrollo).
- one.html: Modo de juego individual contra IA básica.
- js/ Contiene los scripts en JavaScript:
  - `imbatible.js` (Lógica para el modo imbatible, en desarrollo).
  - `javascript.js` (Funciones principales del juego).
  - `one.js` (Lógica para el modo One Player).
  - `pj.js` (Funciones auxiliares).
- style/ Contiene las hojas de estilo:
  - `imbatible.css` (Estilos del modo imbatible).
  - `interfaz.css` (Diseño general del juego).
  - `one.css` (Estilos del modo One Player).
  - `style.css` (Estilos principales).
